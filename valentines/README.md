## Prerequisites

## Getting started

```sh
pnpm install
pnpm dev
```

Open http://localhost:5173.

## Testing

```sh
pnpm test          # single run
pnpm test:watch    # watch mode
```

## Production build

```sh
pnpm build    # outputs to dist/
pnpm preview  # preview the build locally
```
